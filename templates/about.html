Descripcion del proyecto:

AI-Folder, es un proyecto de prueba que intenta ser el lugar de union entre desarrolladores, entusiastas, amantes de la tecnologia o simplemente interesados en aplicaciones de Machine Learning e Inteligencia Artificial.

Tecnologias utilizadas:

  • Python
  • JavaScript
  • Django Framework 4.2
  • HTML5
  • CSS3
  • Bootstrap 5
  • Bootswatch
  • SQLite (Desarrollo)
  • PostgreSQL (Despliegue)

Librerias de Python/Django utilizadas:

  • autopep8 2.0
  • crispy-bootstrap5 0.7
  • django-crispy-forms 2.0
  • asgiref
  • sqlparse 0.4
  • whitenoise 6.4

Control de avance

  • Control de Usuarios con Jerarquias
  • Acceso de Usuarios
  • Registro de Usuarios
  • Crud Favoritos de Usuarios
  • Crud Tareas pendientes de Usuarios
  • Exploracion de Favoritos de Usuarios
  • Busqueda de Favoritos de Usuarios
  • Sistema de mensajes privados entre Usuarios
  • Sistema de comentarios de Favoritos de Usuarios
